# Calc Lang

For my tutorial.

# Grammar

https://amm834.github.io/calacLang/

## Usage

```bash
npm run parse # parse the grammer to generate parser
```

## Simple Program

```
1 + 2
2 + 3 + 8
```

##  Generate AST

```bash
node parser.js example.calc 
```

## Transpile To JS

```bash
node translator.js example.calc.js.json
```


## Run JS

```bash
node example.js
```
